# Extension Icons

You need to create the following icons in the `icons/` directory:

## Required Icons:
- `icon16.png` (16x16px)
- `icon48.png` (48x48px)
- `icon128.png` (128x128px)
- `icon-connected16.png` (16x16px) - Green version
- `icon-connected48.png` (48x48px) - Green version
- `icon-connected128.png` (128x128px) - Green version

## Quick Generation

You can use an online tool like:
- https://favicon.io/
- https://realfavicongenerator.net/
- Canva.com

Or use this simple template:
- Base: Purple/Blue gradient circle
- Symbol: White lock or shield icon
- Connected: Same but with green background
