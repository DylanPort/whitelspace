⚠️ CRITICAL: BEFORE DEPLOYING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. GENERATE FRESH KEYPAIRS
   └─ Old keypairs may be compromised if they were in Git
   
   PowerShell:
   cd node-keys
   for ($i=1; $i -le 10; $i++) {
     solana-keygen new --outfile "bootstrap-node-$i-keypair.json" --no-bip39-passphrase
   }

   Bash:
   cd node-keys
   for i in {1..10}; do
     solana-keygen new --outfile bootstrap-node-$i-keypair.json --no-bip39-passphrase
   done

2. SET SECURE PERMISSIONS
   chmod 600 node-keys/*.json
   chmod 700 node-keys

3. CREATE ENCRYPTED BACKUP
   tar -czf node-keys.tar.gz node-keys/
   gpg --symmetric --cipher-algo AES256 node-keys.tar.gz
   
   Store encrypted backup in 2+ secure locations
   Delete unencrypted tar file

4. VERIFY GIT PROTECTION
   git status
   
   ✓ Should NOT show node-keys/ directory
   ✓ Should NOT show any *.json keypairs
   
   If it does, check .gitignore is properly saved

5. FUND NODES MINIMALLY
   Only 0.01-0.05 SOL per node (just for transaction fees)
   Use separate secure wallet for funding
